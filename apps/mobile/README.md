# OpenLintel Mobile App

Cross-platform mobile application for site capture and project management.

## Key Features

- Guided photo and video capture
- AR measurement tool
- LiDAR integration (iOS)
- Site management (daily logs, checklists, photos)
- Approval workflows
- Push notifications
- Offline mode with sync
