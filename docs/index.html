The Pipeline

Photo to finished home, automated

Nine stages. One continuous pipeline. Every output feeds the next.

01
Room Photo / Floor Plan
Upload photos, scanned floor plans, or DWG files. The system builds a 3D understanding of your space.
02
AI Design Engine
Generates 3 variants at different budget tiers, respecting your constraints and style preferences.
03
Technical Drawings
Produces DWG/DXF floor plans, elevations, and sections — industry-standard formats, not just renders.
04
Bill of Materials
Room-by-room, exact quantities. Every screw, hinge, panel, and fitting — nothing estimated.
05
CNC Cut Lists
Nesting-optimized cut lists with grain direction, edge banding schedules, and waste minimization.
06

As-built drawings, warranty tracking, maintenance schedules, and a digital twin of the finished space.

Architecture

Built on proven foundations

A pragmatic architecture that uses the right tool for each job.

Design philosophy: Use LLM/VLM APIs for everything they are good at. Use open-source tools only where they cannot help — pixel-level vision, 3D geometry, binary file formats, and combinatorial optimization.
AI Layer
VLM/LLM APIs via LiteLLM
Users bring their own API keys. Swap between GPT-4o, Claude, Gemini, or any provider through a unified interface.
Orchestration
LangGraph Agent Orchestration
Stateful, multi-step agent graphs that coordinate across all pipeline stages with built-in retry and human-in-the-loop.
Tools
24 Specialized Open-Source Tools
Purpose-built tools for tasks that LLMs cannot handle: segmentation, depth estimation, 3D reconstruction, file generation, and optimization.
Infrastructure
Temporal + Event-Driven
Durable workflow execution with Temporal, event-driven microservices, and a real-time 3D frontend powered by Three.js.
Key Technologies
SAM 2 Depth Anything V2 COLMAP ezdxf LibreDWG OR-Tools Three.js Temporal LiteLLM LangGraph
